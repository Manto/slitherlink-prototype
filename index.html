Slitherlink

Slitherlink is a challenging logic puzzle where you must draw a single continuous loop.

  • Objective: Create one continuous loop connecting vertices
  • Numbers: Each numbered cell shows how many of its sides must be part of the loop (0-4 for squares, 0-6 for hexagons)
  • Rules: The loop cannot cross itself and must satisfy every numbered cell
  • Left click: Draw a line segment between vertices
  • Right click: Mark with X (no line goes there)
