Slitherlink

Slitherlink is a challenging logic puzzle where you must draw a single continuous loop that connects all dots on the grid.

  • Objective: Create one continuous loop that visits every dot exactly once
  • Numbers: Each numbered cell shows exactly how many of its four sides must be part of the loop
  • Rules: The loop cannot cross itself and must satisfy every numbered cell
  • Strategy: Use logical deduction to determine which lines must be drawn and which spaces must be blocked
  • Left click: Draw a line segment between dots
  • Right click: Mark a space with X (reminding yourself that no line can go there)
